## Learning #6 (Iteration 3) [Pattern]

AlertDialog uses controlled mode with open/onOpenChange props. For delete actions, use variant='destructive' on AlertDialogAction. For multi-target deletes (like media), store the target in state ({assetId, filename} | null) and use the state as the open condition.
