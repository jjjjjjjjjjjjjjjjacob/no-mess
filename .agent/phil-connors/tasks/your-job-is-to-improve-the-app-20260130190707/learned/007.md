## Learning #7 (Iteration 4) [Anti Pattern]

When using useMemo with filter+sort, use const for the filtered array since .sort() mutates in place and returns the same ref. Biome enforces useConst rule. Also: always put react imports after convex/react and lucide-react (Biome organizeImports).
